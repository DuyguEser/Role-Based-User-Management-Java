# Role-Based User Management System (Java)

Role-based user and admin management system developed using Java OOP principles.

## Features
- User login system
- Role-based authorization (User / Admin)
- Object-Oriented Programming principles
- Inheritance and encapsulation

## Project Structure
```
src/
└── usermanagement/
    ├── User.java
    ├── Admin.java
    └── Main.java
```

## How It Works
- Users log in using username and password
- System assigns roles based on credentials
- Admin inherits from User class

- Java
